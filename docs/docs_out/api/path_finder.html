path_finder.app_utensils module¶

path_finder.app_utensils.build_clickable_scheme_html(steps_data: list, route_id: str, is_predicted: bool = False) → str[source]¶

Builds the interactive reaction scheme HTML. Each arrow is clickable and opens a panel with conditions, yield, and copyable SMILES. Predicted routes use orange colouring; purification steps get a dashed brown arrow.

path_finder.app_utensils.build_score_table_html(details: dict, criteria: list, weights: dict, lang: str = 'en') → str[source]¶

Renders the score breakdown table for a route card (raw score, weight, contribution per criterion).

path_finder.app_utensils.build_why_ranked_html(rank: int, score_total: float, details: dict, criteria: list, weights: dict, steps_data: list, lang: str = 'en') → str[source]¶

Builds the ‘Why is this route ranked #N?’ box with one plain-language bullet per top-2 criterion.

path_finder.app_utensils.display_route_card(score_total: float, details: dict, route: dict, criteria: list, weights: dict, all_results: list, rank: int = 1, badge: str = '📚 dataset', lang: str = 'en') → None[source]¶

Renders a full route card in a Streamlit expander: status banner, metrics, score table, why-ranked box, PDF download, reaction scheme, substances list. Expanded by default only for rank 1.

path_finder.app_utensils.get_targets_cached(path: str) → dict[source]¶

Cached {target_name: SMILES} mapping for the molecule selector in the Route Search tab.

path_finder.app_utensils.hires_fig(*args, dpi: int = 180, **kwargs)[source]¶

plt.subplots() wrapper that sets the background to FIG_BG so charts blend with the page.

path_finder.app_utensils.is_purification_step(step: dict) → bool[source]¶

True if step is a purification/isolation (keyword or identity transform).

path_finder.app_utensils.load_banner(path: str) → str[source]¶

Reads a local image and returns a base64 data URI for HTML embedding. Returns “” if missing.

path_finder.app_utensils.load_dataset_cached(path: str) → dict[source]¶

Cached wrapper around rt.load_reaction_dataset() — avoids reloading on every widget interaction.

path_finder.app_utensils.make_comparison_chart(sel_results: list, criteria: list, lang: str = 'en')[source]¶

Grouped horizontal bar chart for side-by-side criterion score comparison in the Analysis tab.

path_finder.app_utensils.make_ranking_chart(results: list, target_name: str, lang: str = 'en')[source]¶

Horizontal bar chart ranking routes by score. Shown above the route cards when more than one route exists.

path_finder.app_utensils.make_yield_chart(steps_route: list, lang: str = 'en')[source]¶

Bar chart of reported step yields. Steps without a yield are simply omitted — no implied 0%.

path_finder.app_utensils.smiles_copy_widget(smiles: str, label: str = '') → None[source]¶

Renders a compact SMILES label with a one-click Copy button using the browser clipboard API.

path_finder.app_utensils.strip_emoji(text: str) → str[source]¶

Strips emoji from text and trims whitespace — matplotlib chokes on Unicode emoji.

path_finder.molecule_rendering.is_trivial_smiles(smiles: str) → bool[source]¶

path_finder.report_builder module¶

path_finder.report_builder.build_route_report_pdf(score_total: float, details: dict, route: dict, criteria: list) → bytes[source]¶

Generates a multi-page A4 PDF for one synthesis route. Page 1: summary header, metric cards, score table, starting material structures. Pages 2+: up to 3 reaction steps per page with conditions and molecule images. Returns raw PDF bytes ready for st.download_button().

path_finder.route_engine module¶

path_finder.route_engine.adapt_route(route: dict) → dict[source]¶

Converts a raw AiZynthFinder route into the internal dict format. Reverses retrosynthetic order, identifies starting materials (tree leaves) and the canonical target SMILES.

path_finder.route_engine.average_yield(steps: list) → float | None[source]¶

Average step yield in percent, ignoring steps with no reported yield.

path_finder.route_engine.bottleneck_yield(steps: list) → float | None[source]¶

Lowest reported step yield in percent, or None if no yields are recorded.

path_finder.route_engine.build_dataset_smiles_index(dataset: dict) → set[source]¶

All canonical SMILES (products and reactants) present in the dataset.

path_finder.route_engine.build_solvent_map(tox_index: dict) → dict[source]¶

Maps common solvent abbreviations and names to SMILES, extended with whatever is already in tox_index.

path_finder.route_engine.calc_atom_economy(reactants_smiles, product_smiles) → float[source]¶

Atom economy = MW(product) / sum(MW(reactants)), capped at 1.0. Returns 0.0 for invalid products.

path_finder.route_engine.calc_e_factor(reactants_smiles, product_smiles, yield_fraction) → float[source]¶

E-factor score = 1 / (1 + waste/product). Missing yield treated as 1.0.

path_finder.route_engine.calc_toxicity_score(reactants_smiles, conditions, tox_index, solvent_map) → float[source]¶

Safety score = 1 - mean(hazard) across reactants and solvents. Unknown compounds default to 0.5.

path_finder.route_engine.compute_all_scores(route: dict, tox_index: dict) → dict[source]¶

Raw scores for all criteria on one route (used by the Analysis tab comparison table).

path_finder.route_engine.compute_atom_economy(route_data: dict, tox_index: dict) → float[source]¶

Mean atom economy across all steps. Returns 0.0 for empty routes.

path_finder.route_engine.compute_e_factor(route_data: dict, tox_index: dict) → float[source]¶

Mean E-factor score across all steps. Missing yields count as 100%.

path_finder.route_engine.compute_steps(route_data: dict, tox_index: dict) → float[source]¶

Score = 1/n_steps. Shorter routes score higher.

path_finder.route_engine.compute_toxicity(route_data: dict, tox_index: dict) → float[source]¶

Mean safety score across all steps. Returns 0.5 (neutral) for empty routes.

path_finder.route_engine.compute_weights(criteria: list) → dict[source]¶

Inverse-square weights (1/i²), normalised to sum 1.

path_finder.route_engine.compute_yield(route_data: dict, tox_index: dict) → float[source]¶

Cumulative yield score. Returns 1.0 for predicted routes (yield is excluded from their scoring).

path_finder.route_engine.cumulative_yield(steps: list) → float[source]¶

Product of all reported step yields as a fraction. Missing yields count as 1.0 (neutral).

path_finder.route_engine.enrich_aiz_route_with_rxninsight(aiz_route: dict, rxni_db, route_index: int) → dict[source]¶

Builds dataset steps for a fully predicted AiZ route using Rxn-INSIGHT for reaction type and conditions. Yield stays None and is excluded from scoring.

path_finder.route_engine.filter_routes_by_starting_materials(aiz_routes, dataset, target_smiles, target_name='') → list[source]¶

Returns dataset routes whose last product matches target_smiles. Falls back to name matching if the SMILES comparison fails. Truncation to top_n happens later in find_best_routes().

path_finder.route_engine.find_best_routes(target_smiles: str, criteria_priority: list, dataset_path: str = 'reaction_dataset.json', toxicity_path: str = 'toxicity_dataset.json', config_path: str = 'config.yml', top_n: int = 3, target_name: str = '', include_predicted: bool = True, rxninsight_db_path: str = '', generic_dataset_path: str = '', n_aiz_routes: int = 25) → dict[source]¶

Full pipeline: loads datasets, runs AiZynthFinder MCTS, classifies routes into dataset/validated/predicted, scores each section and returns the top_n. Raises ValueError if criteria_priority doesn’t contain exactly 3 known criteria.

path_finder.route_engine.fmt_conditions(cond: dict) → str[source]¶

Formats a conditions dict as a readable string: temp · solvent · reagents · apparatus.

path_finder.route_engine.get_all_dataset_routes_for_target(dataset: dict, target_name: str, target_smiles: str = '') → list[source]¶

Returns all dataset routes matching target_name (case-insensitive) and optionally target_smiles.

path_finder.route_engine.get_novel_routes_from_aizynthfinder(aiz_routes, dataset, target_name, rxni_db) → list[source]¶

Legacy wrapper kept for compatibility — prefer process_novel_routes() which also validates against the generic dataset.

path_finder.route_engine.get_reaction_info_rxninsight(reactants: list, product: str, rxni_db) → dict[source]¶

Classifies a reaction and suggests conditions via Rxn-INSIGHT. Without rxni_db: type and class only. With it: also solvent/catalyst/reagent. Returns {} if Rxn-INSIGHT is unavailable or the SMILES are invalid.

path_finder.route_engine.get_substances_list(steps_data: list) → dict[source]¶

Splits a route’s molecules into to_buy, to_prepare, solvents, and reagents.

path_finder.route_engine.get_targets_from_dataset(dataset: dict) → dict[source]¶

Returns {target_name: canonical_SMILES} for each target in the dataset. Tries explicit metadata first, then scans steps in reverse for the largest product (>=15 heavy atoms) as a proxy for the final structure.

path_finder.route_engine.is_route_covered_by_dataset(aiz_route: dict, dataset_smiles_index: set, threshold: float = 0.4) → bool[source]¶

True if >=40% of the route’s products are already in the dataset. That threshold is tight enough to skip genuine duplicates without dropping routes that share only a few intermediates.

path_finder.route_engine.load_generic_reaction_dataset(path: str) → dict[source]¶

Loads individual USPTO reactions for step validation. Returns {} if the file is missing.

path_finder.route_engine.load_reaction_dataset(path: str) → dict[source]¶

Load the curated reaction JSON and build three lookup indexes: by_product, by_reactant, by_route. Accepts a bare list or a {“reactions”: […]} dict. Raises FileNotFoundError if path is missing.

path_finder.route_engine.load_rxninsight_database(path: str)[source]¶

Loads the USPTO Rxn-INSIGHT parquet for condition suggestion. Returns None if missing or not installed.

path_finder.route_engine.load_toxicity_dataset(path: str) → dict[source]¶

Loads hazard scores keyed by canonical SMILES. Returns {} if the file is missing (scores default to 0.5).

path_finder.route_engine.match_step_in_generic_dataset(reactants: list, product: str, generic_ds: dict)[source]¶

Looks up a reaction step in the generic dataset. Tries an exact (reactants + product) key first, then falls back to product only.

path_finder.route_engine.process_novel_routes(aiz_routes: list, dataset: dict, generic_ds: dict, target_name: str, rxni_db) → tuple[source]¶

Processes AiZ routes not already covered by the main dataset. Returns (validated_routes, predicted_routes).

path_finder.route_engine.rank_weighted(routes: list, criteria: list, tox_index: dict) → list[source]¶

Scores and ranks routes using 1/i² weights. For purely predicted routes, yield is excluded and the remaining weights are renormalised to sum to 1. Returns [(total_score, details, route)] sorted descending.

path_finder.route_engine.run_aizynthfinder(target_smiles: str, config_path: str = 'config.yml', n_routes: int = 25) → list[source]¶

Runs AiZynthFinder MCTS and returns up to n_routes adapted route dicts.

path_finder.route_engine.safe_mol(smiles: str)[source]¶

RDKit Mol object, or None if the SMILES is invalid.

path_finder.route_engine.to_canonical(smiles) → str[source]¶

Canonical SMILES via RDKit, or the original string if unparseable, or “” if empty.

path_finder.route_engine.validate_aiz_route_against_generic_dataset(aiz_route: dict, generic_ds: dict, rxni_db, route_index: int) → dict[source]¶

Validates each AiZ step against the generic dataset. Matched steps keep real conditions; unmatched ones get Rxn-INSIGHT predictions. Final status is validated / partial / predicted based on how many steps matched.

path_finder.route_engine.validate_smiles_for_aizynthfinder(smiles: str) → str[source]¶

Validates and canonicalises a SMILES before passing it to AiZynthFinder. Raises ValueError if empty or unparseable.
